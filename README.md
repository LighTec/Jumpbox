# Fall 2019 - CPSC 441 Project

## Requirements
- Java JDK 1.8
## Running in IntelliJ
- Go to ServerMain.java
- Run ServerMain.main() (Ctrl-Shift-F10)
- Go to Main.java
- Run Main.main() (Ctrl-Shift-F10)
### Running Multiple Clients
- Go to Main.java
- Edit Configurations in the toolbar
- Enable 'Allow running in parallel'
